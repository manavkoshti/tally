===========================================================
PROJECT GOAL
===========================================================

I want to build a smart accounting automation web application.

The goal is:

Users should upload invoices or create invoices manually,
and the system should automatically generate accounting entries
like Tally accountant entries.

The system should automatically:
- detect invoice type
- detect GST
- detect party/customer/vendor
- create ledger mappings
- generate debit/credit entries
- create vouchers
- sync entries into Tally Prime automatically

The system should reduce manual accountant work.

This is NOT a simple invoice software.
This is an accounting automation platform.
